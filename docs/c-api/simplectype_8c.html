Detailed Description

Example how to create and use a simple C node type and the high-level API. To understand how to use the high-level C API with existing or custom C types using the generator from JSON (recommended), take a look at highlevel.c

Note that this sample's 'mytype*' uses all features of simplectype, usually some options will not be used in most applications, such as port connections and disconnection events or context. One example of the simplistic version is the 'isodd' that checks if if the given number is odd or even.
