Introduction to the Guidelines

 

This guide is intended to promote consistent usage of the Soletta brand, including, but not limited to, the Soletta name

and the Soletta Logo. Permission is not granted to use or reproduce any of the Soletta branding assets in any manner inconsistent with those represented in these guidelines. Below, we have addressed font usage, color palette, logo variations and scaling, usage of the noun and general usage. If you seek assets or use cases not comprehended in this guideline, please contact us. We will be able to provide guidance for unique requests, or potentially generate an asset to fill your need.

Clear Sans is an OpenType Font,

that can be freely utilized by application, browser and operation system developers.

 

 

Clear Sans has been recognized as a versatile OpenType font for screen, print and web. Clear Sans was designed with on-screen legibility in mind.
